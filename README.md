## 🎯 Sobre

**Coffee & Books** é um projeto web desenvolvido para criar uma experiência imersiva em uma cafeteria literária. O site apresenta um cardápio sofisticado de bebidas, um catálogo de livros recomendados, informações sobre o estabelecimento e um formulário de contato interativo.

Este projeto foi desenvolvido como parte do aprendizado em **desenvolvimento web frontend**, demonstrando habilidades em **HTML5**, **CSS3** e **JavaScript**.

---

## ✨ Funcionalidades

- 🏠 **Página Inicial**: Homepage atrativa com apresentação visual
- ☕ **Cardápio**: Catálogo completo de bebidas com descrições
- 📖 **Livros**: Seção de recomendações literárias
- ℹ️ **Sobre**: Informações detalhadas sobre o estabelecimento
- 📝 **Formulário de Contato**: Interativo com validação
- 📱 **Design Responsivo**: Adaptável para diferentes tamanhos de tela
- ✨ **Interatividade**: JavaScript para melhor experiência do usuário

## 🚀 Como Usar

### Requisitos
- Navegador moderno (Chrome, Firefox, Safari, Edge)
- Nenhuma instalação necessária!

### Passos para Executar

1. **Clone o repositório**
```bash
git clone https://github.com/DudaAngeli/CoffeeAndBooks.git
cd CoffeeAndBooks
```

2. **Abra no navegador**
   - Localize o arquivo `index.html`
   - Abra com duplo clique ou arraste para o navegador
   - Ou use uma extensão como Live Server do VS Code

### Usando Live Server (Recomendado)

Se estiver usando **Visual Studio Code**:

1. Instale a extensão "Live Server"
2. Clique com botão direito em `index.html`
3. Selecione "Open with Live Server"
4. Navegador abrirá automaticamente em `http://localhost:5500`

## 🎨 Páginas

### **Home (index.html)**
Página inicial com design atrativo que apresenta o conceito de Coffee & Books com navegação clara.

**Recursos:**
- Banner visual impactante
- Menu de navegação funcional
- Introdução ao tema

### **Cardápio (cardapio.html)**
Catálogo interativo de bebidas e itens do cardápio.

**Recursos:**
- Lista de produtos com descrições
- Preços e ingredientes
- Design organizado

### **Livros (livros.html)**
Seção de recomendações literárias para acompanhar seu café.

**Recursos:**
- Galeria de livros recomendados
- Autores e sinopses
- Relacionamento temático com café

### **Sobre Nós (about.html)**
Histórico e informações sobre o estabelecimento.

**Recursos:**
- Descrição detalhada
- Missão e valores
- Informações de contato

### **Contato (form.html)**
Formulário para que visitantes possam entrar em contato.

**Recursos:**
- Campos de entrada validados
- Submissão de dados
- Confirmação visual (formAction.html)
